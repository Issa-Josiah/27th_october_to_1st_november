In css we have the selector, the property and the value of the property.

the property and the value are the declaration and put in calibraces

    Type of selectors
  1. Element Selector -> eg p{}
  2. ID selector -> #homeImage{}
  3. Class Selector -> .homePage{}
  4. Pseudo class selector -> a:hover{}
  5. Attribute selector -> a[href=”https://example.com”]{}

Note: you can select more than one element, eg p, h1, h2{}

    Pseudo class selector -> a:hover{}
  1. a:link - a normal, unvisited link
  2. a:visited - a link the user has visited
  3. a:hover - a link when the user mouses over it
  4. a:active - a link the moment it is clicked
