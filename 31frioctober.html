pseudo selectors

After seeing the types of selectors, lets look at the types of pseudo selectors.

  1. a:link - a normal, unvisited link
  2. a:visited - a link the user has visited
  3. a:hover - a link when the user mouses over it
  4. a:active - a link the moment it is clicked

flex

You can make anything use the flex display approach:

  • display:flex;
  • flex-direction: -> defines row(default) or column
  • justify-content-> aligns items horizontally eg space between
  • align-items -> aligns items vertically eg center

grid

You can make anything use the flex display approach:

Display:grid; // define the type of display

column-gap: -> defines column gap eg 50px

row-gap: -> defines row gap eg 50px

justify-content-> aligns items horizontally eg space between

align-items -> aligns items vertically eg cente

media queries

This ideally allows you to make your pages responsive or adapt to the device media type. In this session, we focus on screen sizes.

I limits css styling to particular or based on particular rules

We can split the our rules based on 3 screens:

  1. mobile
  2. Tablets
  3. Large screen

viewpoint

We will add this to the header section of our html file: meta name="viewport" content="width=device-width, initial-scale=1.0" /

Reason:

Gives you the opportunity to control the pages dimensions ans scaling, initial-scale=1.0 sets the initial zoom level
